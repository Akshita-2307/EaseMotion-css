Button Icon Sizes AK

What does this do?

Demonstrates buttons with different icon sizes (small, medium, large) using SVG icons with consistent styling and hover effects.

---

How is it used?

<button class="btn-icon btn-icon--sm"><svg><!-- icon --></svg></button>
<button class="btn-icon btn-icon--md"><svg><!-- icon --></svg></button>
<button class="btn-icon btn-icon--lg"><svg><!-- icon --></svg></button>

Use the `btn-icon--sm`, `btn-icon--md`, or `btn-icon--lg` modifier class to control icon button size.

---

Why is it useful?

Provides a clear pattern for scalable icon buttons ensuring visual consistency across UI components. The size variants make it easy to maintain proportional icon rendering for different contexts like toolbars, navigation, and call-to-action buttons.

Features

- Three predefined size variants (sm, md, lg)
- SVG icon scaling with currentColor
- Hover and focus states
- Works with any SVG icon
- Consistent padding and border-radius per size
- Accessible focus indicators
- Lightweight and themeable
