:user-valid & :user-invalid

Form validation pseudo-classes that only activate after the user has interacted with a field.

8+ characters Contains a number Uppercase letter

How it works

  • :user-valid applies a green border + check icon after the user has interacted with a valid field
  • :user-invalid applies a red border + error icon after the user has interacted with an invalid field
  • Unlike :valid/:invalid, these do NOT fire on page load — only after user interaction
  • Shake animation draws attention to invalid fields
